=== Linux, Cygwin, and OSX ===

Before to be able to compile this binding you should compile and install
OCamlSDL2, and in particular you should not forget to install the
stub headers from OCamlSDL2 with:  make findinstall_h

Then you can run this command in this directory:

$ make

----------------

Install using ocamlfind:
$ make findinstall

-----------------

Uninstall:
$ make finduninstall
